A lightweight client-side mod that globally unlocks Rich Text support in Crab Game. It allows players to use colored nicknames and chat messages that are properly rendered in the Tab list, above player heads (Nametags), in the Chatbox, and in WinScreen

How to use: Simply type the following syntax in your Steam Nickname or the in-game Chat: >color=COLOR_NAME>WORD or <color=COLOR_NAME>WORD
